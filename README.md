# Predictive Analytics – Sales Forecasting

This project focuses on predicting future sales trends using historical data.

## Objective
To build a predictive model that forecasts future sales and visualizes trends for business decision-making.

## Dataset
- Historical monthly sales data (CSV format)

## Tools Used
- Python (pandas, scikit-learn, matplotlib)
- Power BI

## Approach
- Cleaned and preprocessed sales data
- Built a regression model for sales prediction
- Evaluated model accuracy
- Visualized actual vs forecasted sales in Power BI

## Outcome
The model achieved high accuracy and revealed a clear upward sales trend, supporting data-driven forecasting.
